in client folder install these packages :
"axios"
"bootstrap"
"react-bootstrap"
"react-router-dom"


in server folder install these packages :
"cors"
"express"
"multer"
"mysql"
"mysql2"
"nodemon"

Then open up two terminals, then in the first one :
cd client
npm start

Then in the second one :
cd server
nodemon index.js
